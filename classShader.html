Detailed Description

Manages the compilation and linkage of GLSL shaders into a program.

The Shader class encapsulates functionality to create, compile, and link GLSL shaders. It provides methods to add shaders from source code, compile them into a shader program, and bind or unbind the program for rendering operations. The class also handles the cleanup of shader resources.
