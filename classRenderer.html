Detailed Description

Manages the rendering pipeline.

Inherits from ComponentBase to integrate with an application's component system. This class encapsulates the setup and usage of shaders and vertex array objects (VAOs) to render graphics. It provides methods for initializing the rendering resources, updating the rendering logic, and destroying the resources.
